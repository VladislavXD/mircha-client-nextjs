# Font Preview

## Lora (Serif) - для постов

### Regular (400)
Это обычный текст поста. Он должен быть легко читаемым и приятным для глаз. Литературный стиль создает атмосферу качественного контента.

### Medium (500)
**Это немного более акцентированный текст.** Используется для выделения важных моментов.

### SemiBold (600)
**Это полужирный текст** для создания визуальной иерархии в посте.

### Bold (700)
**Это жирный текст** для максимального акцента и заголовков внутри постов.

### Italic
*Это курсивный текст* для добавления эмоциональности и интонации.

### Комбинации
**Жирный *и курсив*** вместе создают выразительный акцент.

## Примеры использования

### Пример поста 1:
Сегодня я хочу поделиться своими мыслями о **важности красивой типографики** в современных приложениях. *Шрифт имеет значение!* 

Когда текст легко читается, пользователь проводит больше времени с контентом. <mark>Это особенно важно для длинных постов.</mark>

### Пример поста 2:
**Три принципа хорошей типографики:**
1. Читабельность превыше всего
2. Визуальная иерархия через размеры
3. *Эмоциональное воздействие через стиль*

### Пример поста 3 (с форматированием):
Обычный текст <b>жирный текст</b> и <i>курсивный текст</i> с <u>подчеркиванием</u> и <s>зачеркиванием</s> плюс <mark>подсветка</mark>!

### Пример поста 4 (со спойлером):
Это открытый текст, а вот [spoiler]<b>секретная информация</b> с <mark>подсветкой</mark>[/spoiler] внутри спойлера!

## Line Height & Spacing

### Плотный текст
Текст без интервалов между строками читается тяжелее. Глаза быстрее устают.

### Нормальный текст (leading-relaxed)
Текст с нормальными интервалами (1.625) читается легко и комфортно.

### Широкий текст (tracking-wide)
Буквы с увеличенным расстоянием (0.025em) создают ощущение простора.

## Рекомендации

1. **Для постов** используйте 17px с leading-relaxed и tracking-wide
2. **Для коротких текстов** можно уменьшить до 15-16px
3. **Для заголовков** увеличьте до 20-24px
4. **Italic** используйте для цитат и эмфазы
5. **Bold** для важных акцентов
6. **Highlight** для критически важной информации
