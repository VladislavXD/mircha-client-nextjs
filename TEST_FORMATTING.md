# Тестирование форматирования текста

## ✅ Реализовано

### 1. Парсинг HTML тегов в `parseEmojiText.ts`
- `<b>`, `<strong>` → bold
- `<i>`, `<em>` → italic
- `<u>` → underline
- `<s>`, `<strike>`, `<del>` → strikethrough
- `<mark>` → highlight (с цветом `rgba(255, 235, 59, 0.3)`)

### 2. Рендеринг в `EmojiText` компоненте
- Создан `SegmentRenderer` для рекурсивного рендеринга
- Поддержка вложенного форматирования (например, жирный + курсив)
- Работает внутри спойлеров
- Работает вместе с эмодзи и упоминаниями

### 3. Поддержка во всех компонентах
- ✅ `CreatePost.tsx` - редактор с кнопками форматирования
- ✅ `EmojiText` - рендеринг форматированного текста
- ✅ `SpoilerSegment` - спойлеры с форматированием внутри
- ✅ `PostCard` - автоматически использует обновленный `EmojiText`

## 🧪 Примеры для тестирования

### Простое форматирование:
```
<b>Жирный текст</b>
<i>Курсивный текст</i>
<u>Подчеркнутый</u>
<s>Зачеркнутый</s>
<mark>Подсвеченный текст</mark>
```

### Вложенное форматирование:
```
<b><i>Жирный и курсив</i></b>
<u><b>Подчеркнутый жирный</b></u>
<mark><b>Жирный с подсветкой</b></mark>
```

### Форматирование + эмодзи:
```
<b>Привет [emoji:0]</b>
<mark>Важно [emoji:1]</mark>
```

### Форматирование + упоминания:
```
<b>Привет [mention:123|John]</b>
<mark>Смотри [mention:456|Jane]</mark>
```

### Форматирование внутри спойлера:
```
[spoiler]<b>Секретный текст</b> с <mark>подсветкой</mark>[/spoiler]
```

### Комбинация всего:
```
Обычный текст <b>жирный</b> и [mention:123|User] с [emoji:0] и [spoiler]<mark>скрытая</mark> <i>информация</i>[/spoiler]
```

## 🔄 Как это работает

### В редакторе (CreatePost):
1. Пользователь выделяет текст
2. Нажимает кнопку форматирования (Bold, Italic, Underline, Strikethrough, Highlight)
3. `document.execCommand` применяет HTML теги
4. `serializeDOM` сохраняет теги в строку
5. При отправке на сервер отправляется строка с HTML тегами

### В карточке поста (PostCard → EmojiText):
1. Получаем текст с HTML тегами из API
2. `parseEmojiText` парсит строку и создает AST (Abstract Syntax Tree)
3. `EmojiText` рендерит AST через `SegmentRenderer`
4. `SegmentRenderer` рекурсивно обрабатывает вложенные элементы

## 🎨 Стили

### Highlight цвет:
- `backgroundColor: rgba(255, 235, 59, 0.3)` - желтый с 30% прозрачностью
- `color: inherit` - текст наследует родительский цвет

### Автоотключение:
- Highlight автоматически отключается при нажатии пробела
- Текст до пробела остается с подсветкой
- Текст после пробела без подсветки

## 🐛 Известные особенности

1. **Toggling формата**: При нажатии кнопки формат включается/выключается
2. **Вложенность**: Форматы могут быть вложены друг в друга
3. **Спойлеры**: Форматирование сохраняется внутри спойлеров
4. **Сериализация**: HTML теги сохраняются в строку и отправляются на сервер
