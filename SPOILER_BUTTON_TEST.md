## Hero UI Button свойства

```tsx
<Button
  size="sm"           // Маленький размер
  color="default"     // Цвет темы по умолчанию
  variant="solid"     // Заполненная кнопка
  className="..."     // Дополнительные стили
/>
```

Можно изменить на:
- `color="primary"` — синяя
- `color="secondary"` — фиолетовая
- `color="success"` — зелёная
- `color="warning"` — оранжевая
- `color="danger"` — красная
- `variant="bordered"` — с границей
- `variant="flat"` — плоская
- `variant="ghost"` — прозрачная

## Возможные проблемы и решения

### Кнопка не появляется при Ctrl+A

**Причина:** Listener `selectionchange` не подписан.

**Решение:** Проверьте в DevTools → Console, нет ли ошибок. Убедитесь, что `useEffect` с `selectionchange` выполнился.

### Кнопка мерцает при выделении

**Причина:** Слишком частые вызовы `updateSpoilerButtonPosition`.

**Решение:** Добавьте debounce:
```tsx
const debouncedUpdate = useMemo(
  () => debounce(updateSpoilerButtonPosition, 100),
  [updateSpoilerButtonPosition]
);
```

### Кнопка остаётся после снятия выделения

**Причина:** `selection.isCollapsed` не проверяется правильно.

**Решение:** Убедитесь, что в `updateSpoilerButtonPosition` есть проверка:
```tsx
if (!selection || selection.isCollapsed) {
  setShowSpoilerButton(false);
  return;
}
```
